# 密码验证实现计划

## 任务目标
实现用户密码验证功能：将前端传递的明文密码进行哈希处理后，与数据库中存储的哈希值进行比对。

## 实施步骤

### 1. 添加密码哈希工具类
- 创建 `PasswordUtil` 工具类
- 使用 `bcrypt` 算法进行密码哈希处理
- 提供方法：
  - `hashPassword(String rawPassword)` - 将明文密码哈希化
  - `verifyPassword(String rawPassword, String hashedPassword)` - 验证密码是否正确

### 2. 修改 UserMapper
- 在 UserMapper 中添加：根据ID查询用户密码哈希值的方法

### 3. 修改认证服务
- 在登录验证时，调用 `PasswordUtil.verifyPassword()` 验证密码
- 不再使用简单的字符串equals比较

## 技术细节

### bcrypt 工作原理
```
注册时：hash = bcrypt.encode("用户密码")
登录时：matches = bcrypt.matches("用户输入密码", hash)
```

### 依赖项
项目已使用 Spring Security，其中包含 bcrypt 支持，无需额外添加依赖。
