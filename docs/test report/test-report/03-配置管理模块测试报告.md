## 1. 测试概述

### 1.1 测试范围

本次测试覆盖FinSight财务智能分析系统的配置管理模块，包括以下接口：

| 接口 | 方法 | 路径 | 功能描述 |
|------|------|------|----------|
| 获取公式列表 | GET | /api/config/formula | 获取公式列表 |
| 创建公式 | POST | /api/config/formula | 创建新公式 |
| 更新公式 | PUT | /api/config/formula/:id | 更新公式信息 |
| 删除公式 | DELETE | /api/config/formula/:id | 删除公式 |
| 验证公式 | POST | /api/config/formula/validate | 验证公式表达式 |
| 获取阈值配置 | GET | /api/config/threshold | 获取阈值配置列表 |
| 创建阈值配置 | POST | /api/config/threshold | 创建阈值配置 |
| 更新阈值配置 | PUT | /api/config/threshold/:id | 更新阈值配置 |
| 获取动态建议 | GET | /api/config/threshold/dynamic-suggestion/:indicatorCode | 获取动态阈值建议 |
| 获取知识库 | GET | /api/config/knowledge | 获取知识库列表 |
| 创建知识库 | POST | /api/config/knowledge | 创建知识库条目 |
| 更新知识库 | PUT | /api/config/knowledge/:id | 更新知识库条目 |
| 删除知识库 | DELETE | /api/config/knowledge/:id | 删除知识库条目 |
| 导出知识库 | GET | /api/config/knowledge/export | 导出知识库数据 |
| 导入知识库 | POST | /api/config/knowledge/import | 导入知识库数据 |
| 获取版本列表 | GET | /api/config/version | 获取模型版本列表 |
| 模拟版本 | POST | /api/config/version/simulate | 模拟版本切换 |
| 切换版本 | PUT | /api/config/version/switch | 切换模型版本 |
